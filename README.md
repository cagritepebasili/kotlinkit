# kotlinkit
Kotlin utilites: Result wrapper, AsyncTask, SuspendTask, etc

## Installation

Gradle dependency:   
```
compile 'com.github.just-4-fun:kotlinkit:0.2'
```

Maven dependency:  
```xml
<dependency>
  <groupId>com.github.just-4-fun</groupId>
  <artifactId>kotlinkit</artifactId>
  <version>0.2</version>
  <type>pom</type>
</dependency>
```
